# git常见操作

- config 配置
    ```
    git config --global user.name "name"
    git config --global user.email "you@example.com"
    ```
- clone
    ```
    git clone [url]
    ```
- status
    ```
    git status -s // 查看文件状态
    git diff // 查看diff

    ```
- push
    ```
    git add [file] // 提交文件缓存
    git add . // 添加所有文件
    git commit -m 'message' // 提交并添加注释
    git commit -a // 跳过提交缓存
    git push // 发布提交
    ```
- pull
    ```
    git pull origin master // 将master分支最新内容拉下并与本地合并
    ```
